# Corpus

Primary narrative materials for Yneva.

- Documents are diegetic or quasi-diegetic.
- Contradictions are expected and preserved.
- Plausibility evaluation terminology is defined externally by HIPF:
  - https://github.com/J-Rosales/historically-informed-plausibility-framework

## Document format

New documents should use **YAML frontmatter** (see `templates/CORPUS_DOCUMENT_HIPF_TEMPLATE.md`).
